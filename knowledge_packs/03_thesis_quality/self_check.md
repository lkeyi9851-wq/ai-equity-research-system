# Thesis Self Check

Before publishing:

- Is each Core Thesis more than a positive fact?
- Does it connect source evidence to a financial line item?
- Does it explain the market expectation gap?
- Does it state valuation or rating impact?
- Does it include an invalidation trigger?
- Are narrative-only points moved out of Core Thesis?

Fail condition:
- If the chain breaks before financial impact, remove from Core Thesis.
