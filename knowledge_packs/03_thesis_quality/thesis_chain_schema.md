# Thesis Chain Schema

```text
thesis_candidate
source_evidence
industry_mechanism
company_exposure
competitive_capture
financial_line_item
market_expectation_gap
magnitude
timing
valuation_or_rating_impact
invalidation_trigger
probability
rating_relevance
```

Minimum Core Thesis:
- source evidence
- financial line item
- expectation gap
- valuation or rating impact
- invalidation trigger

If any minimum field is missing, downgrade to observation.
