# Formula Library

Core formulas:

```text
P/E = Price / EPS

P/FCF = Price / FCF per share

Equity P/FCF = Market Cap / Free Cash Flow

FCF yield = FCF / Market Cap

Enterprise Value = Market Cap + Debt + Minority Interest + Preferred Stock - Cash

EV/EBITDA = Enterprise Value / EBITDA

Implied Price = Forecast metric per share x Selected multiple

Upside / Downside = Implied price / Current price - 1
```

Use rule:
- A formula output is not evidence by itself. It becomes evidence only after source inputs reconcile.
