# Industrial Machinery Trigger Examples

For an industrial machinery company:

```text
Growth trigger:
Downgrade if FY26/FY27 revenue growth loses double-digit profile or consensus EPS revisions turn negative.

Margin trigger:
Downgrade if group gross margin falls below latest actual level or overseas GPM falls materially toward domestic GPM.

Cash trigger:
Downgrade if OCF fails to track net profit or FCF turns negative after working-capital normalization.

Valuation trigger:
Cut confidence if current price, P/FCF, and FCF/share do not reconcile. Cut rating if refreshed valuation shows upside is mostly explained by stale price or external target price.

Industry trigger:
Downgrade if overseas growth is accompanied by receivables / inventory build, price cuts, or dealer financing risk.
```

Use rule:
- Prefer exact thresholds from the source package.
- If exact thresholds are unavailable, use latest actual level as the safety line and label it.
