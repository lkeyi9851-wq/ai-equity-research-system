# Trigger Examples

Bad:

```text
Competition intensifies.
```

Better:

```text
Downgrade if gross margin falls below latest actual level or if overseas gross margin compresses materially toward domestic gross margin, because the margin thesis depends on higher-margin overseas mix.
```

Bad:

```text
Cash flow worsens.
```

Better:

```text
Downgrade if OCF/NPAT falls below 1.0x or if receivables and inventory grow materially faster than revenue, because the cash-quality thesis depends on accounting earnings converting into cash.
```

Bad:

```text
Valuation is no longer attractive.
```

Better:

```text
Cut confidence if refreshed P/FCF is above 16x without EPS or FCF upgrades; cut rating if it rises above 25x or if upside depends mainly on stale price.
```
