# Public Roadmap

This roadmap describes the public, educational version of the project. Private
case packs, live research notes, paid-source traces, and current investment
theses should stay outside the public repository.

## Current Stage

Experimental MVP. The repository demonstrates a local-first research workflow,
agent role design, source gating, memo generation, web-brief intake, and
deterministic quality checks.

The current public version is not production-ready and is not intended to make
investment recommendations.

## Near-Term Priorities

1. Clarify viewpoint-quality criteria.
2. Add a reusable public case-pack template.
3. Separate blind analysis, external anchors, post-mortem learning, and skill
   updates.
4. Improve evidence-quality scoring and source-trust gates.
5. Add synthetic or historical public examples that do not expose private
   research material.
6. Improve setup instructions and runnable smoke tests.

## Longer-Term Direction

- Better company-understanding skill packs.
- Industry signal interpretation with contradiction checks.
- Valuation reasoning that links business drivers to financial assumptions.
- Skeptic review that challenges the core thesis before memo generation.
- Evaluation harnesses that score reasoning quality, not just output format.

## Public Contribution Ideas

- Suggest historical cases with public source trails and known outcomes.
- Improve source-quality and evidence-quality rules.
- Add synthetic examples for common research mistakes.
- Improve setup documentation and cross-platform scripts.
- Propose better tests for thesis transmission, valuation discipline, and risk
  review.
